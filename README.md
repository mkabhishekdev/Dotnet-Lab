# LeadEngineerTraining2025

A 3-month personal training portfolio simulating the responsibilities of a .NET Technical Lead.  
Includes 3 projects with CI/CD, documentation, sprint planning, and RCA samples.

## Projects
- **MockLeadWork-Project1**: ASP.NET Core Web API (EF Core DB-first)
- **MockLeadWork-Project2**: .NET 8 Web API with modern architecture
- **MockLeadWork-FinalProject**: Full-stack microservices simulation

## Support Folders
- `RCA_Samples/` – Root Cause Analysis documents
- `SprintPlans/` – Weekly sprint planning
- `OnboardingGuides/` – Team onboarding examples
- `ProjectBoards/` – Task/project management snapshots
